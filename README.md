# Data Enginnering Study

## 今後のキャリア
### ファーストステップ
**データエンジニア**<br>
* データ基盤やインフラに関する知見を経た上で徐々にレイヤーを上げていく
* エンジニアリングとビジネスのバランスが五分五分の立ち位置が理想

### 最終的
**CDO（Chief Data Officer）**<br>
* サービス全体（プロダクト, セールス, CS）のデータ戦略
